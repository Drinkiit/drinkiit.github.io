Twitter Mention Network Analysis¶

A user can mention other users in his/her tweet, and a tweet can be replied by other users, or retweeted to other users. This notebook walks through an example of visualizing the twitter mention network. The dataset we have contains 331,062 tweets which are related to Obama Care posted by 22,474 users in year 2012. All of the tweets are stored in a json file, and the typical structure of a tweet is shown as below.

{'entities': {'hashtags': ...,        
 'user_mentions': [{'id': 16672662,        
                    'id_str': '16672662',         
                    'indices': [3, 14],          
                    'name': 'Marty Smith',          
                    'screen_name': 'IndyEnigma'}]},           
 'ideology': 'C',
 'text': 'RT @indyenigma: Four hard truths of health care reform ...',
 'user': {'favourites_count': 567688,
          'followers_count': 13167,
          'friends_count': 13822,
          'id': ...},
  ...
}

Notics that the feature "ideology" does not belong to the original twitter dataset, it is learned by the algorithm of a pervious research. These labels are checked by human and the results are quite reliable. In this notebook, the ideology is used to indicate the group of this user. Here "C" and "L" stand for "Conservative" and "Liberty", respectively.

Imports and Pull Out Mentions¶

We'll proceed in steps, beginning with importing and extracting the twitter data. First, we get the ideology of each user based on the ideologies of theirs tweets. And then we pull out the mention networks.

import numpy as np
import json
import zipfile
import itertools
import collections
from sklearn import preprocessing

def twitterGen():
    with zipfile.ZipFile('obamacare_labeled.json.zip') as z:
        with z.open('obamacare_labeled.json', 'r') as f:
            for line in f:
                yield json.loads(line)
                
# group the tweets by user and extract user's ideology
user_ideology_L = {}
for tweet in twitterGen():
    user_id = tweet["user"]["id"]
    if user_id not in user_ideology_L:
        user_ideology_L[user_id] = 0
    user_ideology_L[user_id] += tweet["ideology"].count("L") - tweet["ideology"].count("C")

# pull out the mention network
# the relationship that a user A (source) mentioned another user B (target) is 
# saved in an edge list [(source, target), ...]
user_mention = [(tweet["user"]["id"], mentions["id"])
                for tweet in twitterGen() if tweet["entities"]["user_mentions"] 
                for mentions in tweet["entities"]["user_mentions"]]

Define the Network Based on Edge List¶

Let's move on to building the network based on the edge list we just pulled out before. We use python-igraph package to create our graph and save it as an igraph.Graph object. In total, there are 288,715 edges (mentions) in the graph. Even we can plot this graph, it will be too messy to get any useful information. Actually, graphs of this size are hard to visualize with any algorithm. So instead of plotting the whole network, we only plot the edges (source -> target pairs) which appear 10 more times in our dataset.

import igraph as ig

Edges = [edge for edge, count in collections.Counter(user_mention).items() if count >= 10]

source, target = zip(*Edges)
le = preprocessing.LabelEncoder()
le.fit(list(set(source) | set(target)))

Edges = zip(le.transform(source), le.transform(target))

G = ig.Graph(Edges, directed=True)

Get the node positions, set by the Kamada-Kawai layout for 3D graphs¶

Let's next generate a 3-dimensional layout for our mention graph. A layout is practically a list of coordinates in an n-dimensional space. With the help of python-igraph package, we can easily generate a layout by the Kamada-Kawai algorithm.

N = len(layt)
Xn=[layt[k][0] for k in range(N)]  # x-coordinates of nodes
Yn=[layt[k][1] for k in range(N)]  # y-coordinates
Zn=[layt[k][2] for k in range(N)]  # z-coordinates
Xe=[]
Ye=[]
Ze=[]
for e in Edges:
    Xe+=[layt[e[0]][0],layt[e[1]][0], None]  # x-coordinates of edges
    Ye+=[layt[e[0]][1],layt[e[1]][1], None]
    Ze+=[layt[e[0]][2],layt[e[1]][2], None]
group = []
labels = []
for v in le.inverse_transform(range(N)):
    if v in user_ideology_L:
        if user_ideology_L[v] > 0:
            group.append(1)
            labels.append("id: " + str(v) + " - Liberty")
        else:
            group.append(0)
            labels.append("id: " + str(v) + " - Conservative")
    else:
        group.append(0.5)
        labels.append("Unknown")

          showgrid=False,

data=Data([trace1, trace2])
fig=Figure(data=data, layout=layout)
