Isotonic Regression

Posted on 2017-03-18 | In Statistics |

The problem of isotonic regression on an ordered set is as follows. Given real numbers $\{ y_1, y_2, \dots, y_n \}$, the problem is to find $\{ m_1, m_2, \dots, m_n \}$ to minimize $\sum_{i=1}^n(y_i-m_i)^2$ subject to the restriction $m_1 \le m_2 \le \dots \le m_n$. A unique solution to this problem exists and can be obtained from the “pool adjacent violators” algorithm.
