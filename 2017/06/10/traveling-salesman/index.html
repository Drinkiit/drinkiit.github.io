TSP Tours of the 31 Provincial Capital Cities in China

Posted on 2017-06-10 | In Statistics |

I loved Todd Schneider’s blog post: The Traveling Salesman with Simulated Annealing, R, and Shiny. In this post, I did the similar visualization on China map. Most of my code are almost direct replications of Todd’s, though it may not be obvious to draw the map of China provinces.

The easiest way to create maps in R is to use the maps package, which is exactly the package used by Todd. The maps package comes with the world map and the map of United States. However, there is no map of China in the package of maps, which means we need to find geographic data file for China. There are two ways to do this, (1) Using mapdata package. Unfortunately, the China map in this library is fairly low-resolution and a little out of date. In mapdata, Chongqing has not been separated from Sichuan province. (2) Downloading some GIS shapefile data for China and then using one of the spatial packages to plot that in R. I downloaded the administrative boundaries from gadm.org. But the shapefile is pretty large and it takes an unreasonable amount of time to plot. So we had better to speed up our map plotting if we want to show it in a shiny app. And I found this question quite useful: How to speed up the plotting of polygons in R?
